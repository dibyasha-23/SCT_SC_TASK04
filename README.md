# Keylogger Program

## Task 04 - Cybersecurity Internship (SkillCraft Technology)

### Description
This project implements a basic keylogger using Python.

It records keyboard inputs and saves them into a text file.

### Features
- Captures key presses
- Saves keys into log.txt
- Stops when ESC key is pressed

### Requirements
- Python
- pynput library

### How to Run
1. Install pynput:
   python -m pip install pynput

2. Run the program:
   python keylogger.py

3. Type on keyboard

4. Press ESC to stop

### Output
- log.txt file containing recorded keystrokes
